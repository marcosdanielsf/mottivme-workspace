# Comandos Claude Code

## Organização
| Comando | Descrição |
|---------|-----------|
| `/organizar` | Analisa e organiza arquivos soltos |
| `/novo-projeto` | Cria estrutura para novo cliente |
| `/status` | Mostra resumo do workspace |

## Workflows n8n
| Comando | Descrição |
|---------|-----------|
| "Qual node para X?" | Busca o node certo |
| "Valide esse workflow" | Verifica erros |
| "Crie workflow para Y" | Gera JSON completo |

## Dicas
- Digite `/` para ver todos os comandos
- Arraste arquivos JSON para o chat para analisar
- Use `/organizar` sempre que a pasta ficar bagunçada

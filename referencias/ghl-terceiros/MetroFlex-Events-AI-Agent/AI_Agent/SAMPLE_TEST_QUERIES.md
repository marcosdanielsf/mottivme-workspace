## Category 1: NPC Division Rules & Weight Classes

### Men's Bodybuilding
```
What are the weight classes for Men's Bodybuilding?
```
**Expected Response**: Should list all 7 weight classes from Bantamweight (up to 143.25 lbs) through Super Heavyweight (over 243.25 lbs)

```
Tell me about the judging criteria for Men's Bodybuilding
```
**Expected Response**: Muscularity, symmetry, proportion, conditioning, presentation

```
What should I wear for Men's Bodybuilding?
```
**Expected Response**: Posing trunks (solid color, no logos, 50% glute coverage)

---

### Classic Physique
```
I'm 5'10" and 185 lbs. Can I compete in Classic Physique?
```
**Expected Response**: Yes, for 5'10"-5'11" height class, max weight is 202 lbs. You're within the limit.

```
What's the difference between Classic Physique and Men's Physique?
```
**Expected Response**: Classic Physique emphasizes size, symmetry, V-taper; Men's Physique focuses on marketability, conditioning, overall aesthetic

```
What are the posing requirements for Classic Physique?
```
**Expected Response**: 7 mandatory poses, 60-second posing routine with music

---

### Men's Physique
```
What are the height classes for Men's Physique?
```
**Expected Response**: 5 height classes (A through E), ranging from under 5'7" to over 6'1"

```
Do I need to do a posing routine in Men's Physique?
```
**Expected Response**: No routine, only comparisons and individual quarter turns

```
What kind of board shorts should I wear?
```
**Expected Response**: Just above the knee, solid color or simple patterns, no branding/logos

---

### Women's Divisions

```
What are the divisions for women?
```
**Expected Response**: Women's Bodybuilding, Figure, Bikini, Wellness, Women's Physique

```
Tell me about the Bikini division
```

## Category 2: Competition Procedures

### Registration
```
How do I register for a MetroFlex event?
```
**Expected Response**: Online via website links, in-person at venue, by mail. Requires name, division, contact info, emergency contact.

```
Do I need an NPC card to compete?
```
**Expected Response**: Yes, NPC membership card required. Available at npcnewsonline.com or purchase at event (higher fee).

```
What happens if I register late?
```
**Expected Response**: Late registration fees apply. Specific amounts vary by event.

### Posing Music
```
How do I submit my posing music?
```
**Expected Response**: Required for Classic Physique, Women's Bodybuilding, Figure, Women's Physique. Max 60 seconds. Submit by deadline (usually 1 week before). MP3/WAV format.

```
What happens if I forget my posing music?
```
**Expected Response**: Should contact event organizer immediately (brian@metroflexgym.com / 817-465-9331)

---

### Competition Day
```
What should I bring on competition day?
```
**Expected Response**:
- Required: NPC card, competitor number, posing suit, tanning products, oil, music (if applicable), ID, water
- Recommended: Snacks, towels, robe, makeup, hairspray, emergency contact

```
What's not allowed backstage?
```
**Expected Response**: Glass containers, outside food in certain areas, unauthorized persons

```
What time should I arrive?
```
**Expected Response**: Check-in times vary by event, typically 2-3 hours before show. Check specific event details.

## Category 7: First-Time Competitors

### Preparation Timeline
```
This is my first competition. Where do I start?
```
**Expected Response**: 10-step guide: 1) Choose division (12-16 weeks out), 2) Select event, 3) Get NPC card, 4) Hire coach, 5) Create meal plan, 6) Start training, 7) Order posing suit, 8) Practice posing, 9) Register, 10) Prepare for show day

```
How long does competition prep take?
```
**Expected Response**: 12-16 weeks typical for first-timers, 8-12 weeks for experienced competitors

```
Do I need a coach?
```
**Expected Response**: Highly recommended, especially for first competition. Coach helps with diet, training, posing, tanning, strategy.

## Response Quality Rubric

### Excellent Response (5/5)
✅ Accurate information from knowledge base
✅ Specific numbers/dates/details
✅ Includes relevant MetroFlex legacy reference
✅ Clear call-to-action or next steps
✅ Professional tone, 2-4 paragraphs
✅ Responds in <2 seconds

### Good Response (4/5)
✅ Accurate information
✅ Mostly specific details
⚠️ Missing legacy reference or CTA
✅ Appropriate length and tone

### Needs Improvement (3/5)
⚠️ Somewhat generic answer
⚠️ Missing specific details
⚠️ Too verbose or too brief
✅ Core information correct

### Poor Response (1-2/5)
❌ Inaccurate information
❌ Hallucinated details not in knowledge base
❌ Completely off-topic
❌ Inappropriate tone
